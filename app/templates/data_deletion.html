User Data Deletion

You can request deletion of data associated with your use of this application, including data stored from Instagram messaging webhook events.

How to request deletion

  1. Email us a deletion request.
  2. Include the following so we can locate and verify the correct records:
    • Your Instagram username (preferred) and/or the Instagram user ID shown in the app (if available)
    • Any conversation identifier visible in the app (if available)
    • Approximate date/time range of messages
  3. If we need additional verification, we may ask follow-up questions to confirm the request.

What we delete

  • Stored DM conversations/messages received via Instagram webhooks and/or synced via the Graph API (where applicable).
  • Related metadata stored for conversation display and automation (for example, delivery status/errors).
